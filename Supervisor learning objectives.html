Workflow Overview - TrainABA Supervision Curriculum Learning Objectives

Note: Selections of this module result in Type-II BACB Continuing Education Units.

    The following learning objectives account for key basic workflow competencies necessary for supervisors and supervisees to successfully conduct supervisory activities as outlined in the TrainABA Supervision Curriculum book series (volumes 1-3) for the purpose of satisfying BACB credentialing and supervision requirements:

  1. The BACB-approved supervisor facilitates ongoing independent fieldwork of BACB-exam candidates using curriculum found in Volume 1: BCBA Reference Manual & Volume 2: BCBA Independent Fieldwork of the TrainABA books.
  2. The registered BACB-exam candidate participates in ongoing independent fieldwork curriculum as delineated in Volumes 1 & 2
  3. The BACB-approved supervisor facilitates Registered Behavior Technician (RBT) exam candidacy requirements to assist candidates in obtaining credentialing using curriculum found in Volume 3 - RBT Credential Workbook and elsewhere on www.trainaba.com.
  4. The BACB-approved supervisor facilitates ongoing supervision to active and non-active RBTs according to current BACB standards in conjunction with the Volume 3 book and supplementary supervision materials found on www.trainaba.com.

Format for each learning objective

    The following structure is salient across learning objectives:

  • Each lesson lasts 5-13 minutes in length
  • Description of skills to be trained
  • Rationale (assists generalization)
  • Behavioral Skills Training elements as appropriate (M.I.R.F. = Model, Instructions, Rehearsal, Feedback)
  • Narrated screen captures, slideshares, and video presentations as appropriate
  • Knowledge checks

Expansion of Objective 1

For the Supervisor

    In each 2-week supervision period, the supervisor needs her own set of books to access the meeting agendas, refer to the task list items of focus for the period, and identify the following key knowledge and performance of the key task list items for that period:

  • Definitions of items in Reference Manual
  • Assessment criteria from Reference Manual for the supervisee performance and/or knowledge check
  • Examples illustrating task list item in Reference Manual to evaluate written examples in each supervisee/exam candidate’s Independent Fieldwork book
  • Peer-reviewed literature germane to the task list in Reference Manual
  • Write in the supervisee/exam-candidate’s book for that individual’s running 4th Edition Task List assessment (Knowledge demonstration as well as performance measures required)
  • Review and follow meeting agendas in the Independent Fieldwork book

For the Supervisee/Exam Candidate

    In each 2-week supervision period, the supervisee/exam-candidate needs her own set of books for the following tasks:

  • Define terms when indicated by blank workbook box (writing definitions in Independent Fieldwork book and checking with definitions in Reference Manual)
  • Practice Assessment criteria from Reference Manual to demonstrate knowledge and/or performance measures of targeted task list items for that period
  • Receive updates from supervisor for 4th Edition Task List Assessment written in Independent Fieldwork book
  • Review examples in Reference Manual for each task list item and generalize with a permanent product by writing exam candidate’s own examples/non-examples in the book following the biweekly progression of task list items identified for each 2-week period.
  • Prior to Exam: Review personal Assessment measures to brush up on items as recommended by supervisor
  • Prior to Exam: Review written items in Independent Fieldwork book and compare with Reference Manual to practice application and knowledge before testing period
